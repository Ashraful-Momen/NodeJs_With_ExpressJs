## **1️⃣ Install Node.js (on Ubuntu/Debian VPS)**

**Option A: Using NodeSource (recommended for latest LTS):**

```bash
# Update system
sudo apt update && sudo apt upgrade -y

# Install NodeSource repo (example for Node 20 LTS)
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

# Install Node.js
sudo apt install -y nodejs

# Check versions
node -v
npm -v
```

**Option B: Using nvm (Node Version Manager, optional)**

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.6/install.sh | bash
source ~/.bashrc
nvm install 20
nvm use 20
node -v
npm -v
```

---

## **2️⃣ Initialize a Node.js project**

```bash
# Create a project folder
mkdir my-express-app
cd my-express-app

# Initialize Node.js project
npm init -y
```

* This creates `package.json`.

---

## **3️⃣ Install Express.js**

```bash
npm install express
```

* Optional (for dev dependencies):

```bash
npm install --save-dev nodemon
```

* Your `package.json` will include Express as dependency.

---

## **4️⃣ Create a simple Express app**

`index.js`:

```javascript
const express = require('express');
const app = express();
const PORT = process.env.PORT || 3000;

app.get('/', (req, res) => {
  res.send('Hello, Express with Node.js!');
});

app.listen(PORT, () => {
  console.log(`Server is running on port ${PORT}`);
});
```

---

## **5️⃣ Test locally**

```bash
node index.js
```

* Visit `http://<VPS-IP>:3000` → should show `Hello, Express with Node.js!`
