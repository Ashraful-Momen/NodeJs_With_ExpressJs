**Auto-reload Node.js on code changes → use `nodemon`**

1. **Install** (in your project folder):
   ```bash
   npm install --save-dev nodemon
   ```

2. **Run**:
   ```bash
   npx nodemon app.js
   ```

✅ Now, any code change auto-restarts the server.  
⚠️ Only for development — not for production.
